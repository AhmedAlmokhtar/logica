iNTERACTIVE SCREENS

The iTable provides the mechanisms for users to more readily integrate group experiences and knowledge through easy to use touch based interactions, and the responsive, multi-touch user interface means users can fully engage with data, allowing critical information to be distilled, compressed or amplified during a discussion.

Coupled with the software applications, the iTable gives users the ability to seamlessly share on-screen visualizations and interactions over the network in a common work space. This means that disparate groups of users on different iTables can see, interact with and share information as if they were all gathered around the same system, creating a common operating picture.

 

Benefits

A responsive, multi-touch user interface allows user to fully engage with data, allowing critical information to be distilled, compressed or amplified during a discussion.

Structures discussion geospatially, supporting a common language across a hierarchy of situational groups (i.e. decision makers, knowledge experts, information gathers and responders).

Large display size provides a natural social convention where users can gather around, eye to eye and discuss a situation.

Integrated collaboration software creates a shareable work environment for real-time distribution of information across local and remote users.

Expanding on the concept of traditional conference or map tables, iTables facilitate seamless integration of static and dynamic project files.

Customized stand facilitates convenient deployment in a wide range of environments.
